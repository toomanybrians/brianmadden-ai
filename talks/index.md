Evening keynote introducing the workspace-as-control-plane thesis. AI agents and human workers share the same workspace. Covers computer-using agents, OSWorld benchmarks, the Windows-is-dead running joke, and why this usage pattern will last longer than anyone expects. Full transcript available.

Ten reasons why AI agents will enter the enterprise through Windows desktops. Agentic AI will use the same tools humans use. Covers Windows Agent Arena, Claude computer use, Microsoft Recall, and why the desktop running in the cloud is the entry point for AI. Full transcript available.

**Microsoft Recall: A New Era for Workplace AI** (2024-06-06)

**It's Springtime for AI** (2024-03-26)
